Badge

Assign and sort items with a 1 to 5 rating system displayed as an icon badge

Evaluation displayed on the upper right of the icon.

Function

Context commands to manage file/folder ranking for sorting and searching

Note

  • Steps to use:
    1. Select one or more items in a file list
    2. Right click the selection to show the Badge menu
    3. Choose:
      • 0 to 5: Assign to this rank
      • Open: Search for items with this rank and list them in a tab
      • Sort: Sort the file list by rank
  • File/folder rank is displayed as an icon overlay.
