Flat

Flat view addon

Tab context command to show all items under the folder branch as a flat list

Description

Flat view addon recursively "squashes" the view of a certain folder so that all files contained in it are displayed in a single list, regardless of what subfolder the are located in.

This may be useful if you quickly want to see all files and subfolders contained in a certain directory, without having to manually enter every single one of them.
