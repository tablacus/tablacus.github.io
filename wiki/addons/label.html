Label

Function

Context menu commands to assign and manage user-defined file/folder labels for sorting and searching purposes

Usage

Right-click on any file or folder and choose an option from the Label sub menu

  • Edit to edit an item's labels
  • Arrange by:label to sort items according to labels
  • Add to add labels from a list of available previously assigned labels
  • Remove to remove any assigned labels

Searching by label

To search for files/folders with a specific label, input label: and then the label text in the address bar. Alternatively (and more easily) use the “Label button” add-on.

Options

Under Tools > Options > Add-ons > Label > General you can tick the Portable option to make label entries assigned to items in the same drive Tablacus is installed work portably, regardless of assigned drive letter.
