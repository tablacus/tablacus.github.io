Go Home

Allows you to quickly navigate back to a home folder by pressing a configurable hot key (by default Alt+ Home). You can also right-click on the tab and from the context menu chose Go Home option to navigate to the previously defined home path.

To set the home folder for a certain tab use the Set Home addon, right-click on the tab and from the context menu chose Set Home option to set current path as home.
