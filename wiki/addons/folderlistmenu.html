Folder list menu

Function

Show a menu of folders listed in a text file

Settings

When installing folder list menu "Folder list menu" is added to type.

Specify a text file in which the folder list is stored as an option.

Basic

menu.cfu

C:\Program Files
C:\Windows
C:\Windows\System32

Display name

menu.cfu

C:\Program Files
#EXTINF:-,Windows folder
C:\Windows
System folder    C:\Windows\System32

Hierarchical menu

menu.cfu

C:\Program Files
C:\Program Files
#EXTINF:-,Windows/Windows folder
C:\Windows
Windows/System folder    C:\Windows\System32

Splitter

menu.cfu

C:\Program Files
-
#EXTINF:-,Windows/Windows folder
C:\Windows
Windows/System folder    C:\Windows\System32
