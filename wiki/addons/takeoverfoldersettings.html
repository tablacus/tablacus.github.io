Take over folder view settings

Same view to all file folders

  • when you navigate, and
  • across TE sessions

Note

  1. By default, TE may change the view unexpectedly when you navigate from a file folder to a system folder (such as “Computer”) and then back. This add-on solves this problem.
  2. By default, TE applies the view in Windows Explorer when it starts. This add-on lets TE apply the last used view in TE.

Config file

takeoverfoldersettings.txt
