Full path bar

Function

Display the full path of:

  • the current folder (when no item is selected),
  • or the current selection

Note

You can opt to display on the title bar. Then you also need to:

  • Set the location to “outside of the TE window”
  • Disable the “title bar” add-on

If multiple items are selected, only the full path of the last selection will be displayed.
