Remember folder view settings

  • Remember the view of each folder when you browse, like the default behavior of Windows.

Options

Config file

remember.xml

Advanced

Options under advanced determine how properties columns references are saved. It is only relevant if you use Tablacus portably on different computers, share configurations across in different environments, or use it under multiple operating systems, otherwise it makes little difference.

  • Normal
  • Path
  • Key

Normal

Save columns by Name.

Example

  • Size
  • Date modified

Pros

Simple

Cons

Changes for different languages.

English German
Size Größe
Date modified Änderungsdatum

Path

Save columns by Path (Cannonical Name).

Example

  • System.Size
  • System.DateModified

Pros

It's same even if language is different.

Cons

It's different for each OS like Windows XP / Vista or later.

Vista or later XP
System.Size Size
System.DateModified Write

Key

Save columns by Key (Property key).

Example

  • {B725F130-47EF-101A-A5F1-02608C9EEBAC} 12
  • {B725F130-47EF-101A-A5F1-02608C9EEBAC} 14

Pros

It's the same in different environment.

Cons

It looks long and complicated.
