Rename the Tab

Change the tab title by assigning a custom name on a per folder basis

Function

  • Display a specific name, instead of the folder name, on the tab header for a specific folder.
  • Names are saved per folder path, so if you navigate away from the current folder you assigned a name in, the name will change.
  • Names will be restored once the path matches a saved folder.
  • For fixed tab names see Fixed Tab Name addon

Usage

  • Right-click on a tab header and click Change Tab Name...

Config file

  • tabname.xml

Japanese

タブに名前を付ける

  • フォルダ名ではない、任意の名前をタブに付ける。
