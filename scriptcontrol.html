Tablacus Script Control 64
  • 64ビット版のWindowsではそもそもMSScript.ocxが存在しないので、代わりにScriptControlを動作するようにします。
インストールとアンインストール

適当なフォルダに解凍してsetup.exeを実行してください。
「Install」でインストール、「Uninstall」でアンインストールを行います。

使い方（Microsoft Script Controlと同じです）

ex1.js 
var SC = new ActiveXObject("ScriptControl");
SC.Language = "VBScript";
SC.ExecuteStatement('MsgBox 1+2');
ex2.js 
var SC = new ActiveXObject("ScriptControl");
SC.Language = "VBScript";
WScript.Echo(SC.Eval("1+2"));
ex3.js 
var SC = new ActiveXObject("ScriptControl");
SC.Language = "JScript";
SC.AddObject("WScript", WScript);
SC.AddCode('function fnx(a,b,c){ WScript.Echo(a + ":" + b + ":" + c) }');
SC.Run("fnx", 1, "data2", "data3");
ex4.js 
var SC = new ActiveXObject("ScriptControl");
SC.Language = "JScript";
SC.AddObject("WScript", WScript);
SC.AddCode('function fnx(a,b,c){ WScript.Echo(a + ":" + b + ":" + c) }');
SC.CodeObject.fnx(1,2,3);
ex5.vbs 
Set SC = CreateObject("ScriptControl")
SC.Language = "VBScript"
MsgBox SC.Eval("1+2")
